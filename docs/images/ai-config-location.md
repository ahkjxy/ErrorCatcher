# AI 配置界面位置说明

## 访问路径

```
管理后台首页
  ↓
点击左侧导航 "设置"
  ↓
在设置页面左侧菜单中选择 "AI 分析配置"
  ↓
AI 配置管理界面
```

## 界面结构

### 左侧导航菜单
```
设置
├── 个人中心
├── 环境配置
├── AI 分析配置 ← 这里！（仅管理员可见）
└── 用户管理（仅管理员）
```

### 主要区域

#### 1. 配置状态卡片
显示当前 AI 功能的状态：
- 启用状态（已启用/未启用）
- 当前模型（如：gpt-4o-mini）
- API Key 状态（已配置/未配置）

#### 2. OpenAI 配置表单
包含以下字段：
- **OpenAI API Key**（必填）
  - 密码输入框
  - 显示当前配置预览
  
- **模型**（必填）
  - 下拉选择框
  - 选项：gpt-4o-mini（推荐）、gpt-4o、gpt-4-turbo、gpt-3.5-turbo
  
- **API URL**（可选）
  - 文本输入框
  - 默认：https://api.openai.com/v1

#### 3. 操作按钮
- 保存配置
- 测试连接
- 重置

#### 4. 使用说明
蓝色提示框，包含：
- 功能说明
- 使用步骤
- 获取 API Key 的链接

## 权限要求

- 只有管理员（admin）角色可以看到和访问此页面
- 普通用户（developer、viewer）无法看到此菜单项

## URL 路径

```
http://localhost:3000/settings
```

访问后，在左侧菜单中选择 "AI 分析配置"

## 相关页面

### 使用 AI 分析的页面

配置完成后，可以在以下页面使用 AI 分析功能：

1. **错误详情页面**
   - 路径：`/errors/:errorId`
   - 位置：点击 "AI 分析" 标签
   - 操作：点击 "开始 AI 分析" 按钮

2. **Dashboard 页面**
   - 路径：`/dashboard`
   - 位置：AI 功能入口卡片
   - 操作：点击查看 AI 分析功能介绍

## 界面截图说明

- 显示一个绿色或灰色的状态标签

- 三个输入字段垂直排列

## 常见问题

**Q: 为什么我看不到 "AI 分析配置" 菜单？**  
A: 只有管理员用户才能看到此菜单。请确认您的账号角色为 admin。

**Q: 配置保存后多久生效？**  
A: 立即生效。保存后可以直接使用 AI 分析功能。

**Q: 如何验证配置是否正确？**  
A: 点击 "测试连接" 按钮，系统会发送一个测试请求验证配置。

**Q: 可以修改已有的配置吗？**  
A: 可以。在表单中输入新的值，点击保存即可更新配置。

**Q: API Key 会显示在界面上吗？**  
A: 不会。出于安全考虑，只显示前7位和后4位，中间用省略号代替。
